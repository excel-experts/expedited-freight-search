üìã CSV Format Requirements

Your CSV file should include the following columns:

  • order_id - Unique identifier from SuperDispatch
  • origin_city - Origin city and state (e.g., "Los Angeles, CA")
  • destination_city - Destination city and state (e.g., "New York, NY")
  • carrier - Carrier name
  • price - Shipment price (numeric)
  • distance - Distance in miles (numeric)
  • order_date - Order date (YYYY-MM-DD format)

Note: price_per_mile will be automatically calculated if not provided.
